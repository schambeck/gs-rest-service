== Spring Native

This guide walks you through the process of creating a `Spring Native` version of "`Hello, World`" RESTful web
service with Spring.

== Package, Run and Test it through Make

Package:

====
[source,bash]
----
make dist
----
====

Run:

====
[source,bash]
----
make run
----
====

Test:

====
[source,bash]
----
make test
----
====
